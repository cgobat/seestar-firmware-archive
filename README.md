# Seestar firmware update packages

This repository contains (in the release section) an archive of over-the-air (OTA) update packages from the Seestar (S50, but I think they're the same) smart telescope.

The contents of these update packages are copyright ZWO&mdash;I make no claims of ownership. Because of Seestar and ASI Air's inclusion/use of GPL'd code, they are obligated to release their source code anyway so it should be no problem to share these here.
